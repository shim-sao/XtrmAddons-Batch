MySQLBatchBackup

MySQLBatchBackup is a series of scripts designs for backing up a remote UNIX database on a local WINDOWS server with SHH and SFTP protocols.

1 - Prerequisites

  • Have Putty to be installed.
  • Download 7-Zip or have to be installed.

Optional :

  • Have MySQL 5+ to be installed.

2 - Configure the application

  • Open the file application.cmd with a text editor.
  • Check and change the folder path MBB_PATH_ZIP where is the software installation of 7-Zip or place the executable in the directory named "zip".
    Comment on adding to the Windows environment variable PATH if another folder path already exists.
  • Uncomment and check the folder path where the Putty software installation is located if it is not already in the Windows environment variable PATH.
  • Uncomment and check the folder path where the MySQL 5+ software installation is located if it is not already in the Windows environment variable PATH.
  • Save.

3 - Configurer un backup ?

Configure the servers and database :

  • In the directory config Copy/Paste the file config.example.cmd and rename it for example : config.monsite.cmd
  • Open the file config.monsite.cmd with a text editor and edit the following informations :
    dbSrc = remote server. Example : 1&1
    dbDest = local serveur.
  • Save.

Create a launch file :

  • Copy/Paste the file backup-example.bat and rename it for example : backup-monsite.bat
  • Open the file backup-monsite.bat with a text editor.
  • Edit the following lines :
  • MBB_CONFIG_FILE : Replace the file name with the one created previously.
  • MBB_PARAM_BACKUP : Enable or disable automatic import into the local database (yes|no).
    Notice, to do this MySQL 5+ must be available.
  • Save.

4 - Start the backup

  • Double click on backup-monsite.bat and follow the instructions.
