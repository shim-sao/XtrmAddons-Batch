This script use 7-Zip :
7-Zip is a file archiver with high compression ratio.
7za.exe is a standalone command line version of 7-Zip.

Source code of 7za.exe and 7-Zip can be found at http://www.7-zip.org/

Place here your downloaded version of 7-Zip
alternative
Change MBB_PATH_ZIP path in the application.cmd script with the path of your downloaded version of 7-Zip
