OpensslBatchCertificate

OpensslBatchCertificate est un script conçu pour créer un certificat SSL avec OpenSSL et Windows.

1 - Prérequis

  • IMPORTANT: Pour ajouter OPENSSL_CONF de manière permanente aux variables d'environnement Windows, ce script doit être exécuté en tant qu'administrateur.

2 - Configurer l'application

  • Ouvrir le fichier openssl-certificate.cmd avec un éditeur de texte.
  • Remplacez la variable HOSTNAME par le nom d'hôte à associer au certificat lors de la création.
  • Vérifier le chemin du dossier APACHE_ROOT où se trouve l'installation du logiciel Apache si celui ci ne se trouve pas déjà dans la variable d'environement Windows PATH.
  • Sauvegarder.

4 - Lancer la génération d'un certificat.

  • Double cliquer sur openssl-certificate.bat et suivre les instructions.
